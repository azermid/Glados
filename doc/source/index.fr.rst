*************
Introduction
*************

Le MultipleLanguageLanguage (MLL) est un langage de programmation réalisé dans le cadre du projet Glados d'Epitech. C'est un langage
qui possède essentiellement la même syntaxe que C, mais qui permet d'écrire du code dans plusieurs langues telles que le français, l'anglais ou le portugais.
