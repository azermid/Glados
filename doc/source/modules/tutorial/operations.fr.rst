Boucles While et For
#####################

Boucle While
=============

La boucle while est utilisée pour exécuter un bloc de code tant que la condition est vraie. La syntaxe est la suivante :

.. code-block:: c

    // Anglais
    while (condition) {
        // code à exécuter
    }

    // Français
    tant que (condition) {
        // code à exécuter
    }

    // Portugais
    enquanto (condição) {
        // código a executar
    }

Exemple
-------

.. code-block:: c

    int i = 0;

    // Anglais
    while (i < 5) {
        i++;
    }

    // Français
    tant que (i < 5) {
        i++;
    }

    // Portugais
    enquanto (i < 5) {
        i++;
    }

Boucle For
===========

La boucle for est utilisée pour exécuter un bloc de code un nombre spécifique de fois. La syntaxe est la suivante :

.. code-block:: c

    // Anglais
    for (initialization; condition; increment) {
        // code à exécuter
    }

    // Français
    pour (initialisation; condition; incrémentation) {
        // code à exécuter
    }

    // Portugais
    para (inicialização; condição; incremento) {
        // código a executar
    }

Exemple
-------

.. code-block:: c

    // Anglais
    for (int i = 0; i < 5; i++) {
        // code à exécuter
    }

    // Français
    pour (int i = 0; i < 5; i++) {
        // code à exécuter
    }

    // Portugais
    para (int i = 0; i < 5; i++) {
        // código a executar
    }
