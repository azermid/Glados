*************
Introdução
*************

O MultipleLanguageLanguage (MLL) é uma linguagem de programação desenvolvida no contexto do projeto Glados da Epitech. É uma linguagem
que possui essencialmente a mesma sintaxe do C, mas permite escrever código em várias línguas, como francês, inglês ou português.
